二、定义

  • 测试应用：基于移动互联网，提供销售推广、代收货款、技术服务的综合性平台。
  • 测试应用服务：指测试应用平台向其注册用户提供的服务，包括但不限于销售推广服务、代收货款服务、便民服务、技术服务等。乙方有权根据甲方提交的资料及甲方的使用记录等单方决定向甲方开发测试应用服务的部分或者全部服务，甲方使用相关服务时，应遵守乙方就相关服务的规定。
  • 用户：注册测试应用账户，使用全部或者部分测试应用服务的注册用户。 商户：申请入驻测试应用、使用测试应用销售推广服务或代收货款服务的用户。
  • 虚假商户：不真实商户，包括但不限于以虚假资料、冒用其它商户的资料申请商户。
  • 测试应用硬件：读取银联卡信息并进行加密处理，与甲方测试应用客户端配合使用实现代收款服务的终端，又称移动POS机，简称硬件。
  • 服务费率：甲方使用测试应用代收货款时承担费用的标准，简称费率。 服务费：甲方通过测试应用普通代收货款功能时应承担的费用。
  • 清算机构：与乙方合作为测试应用用户提供资金结算服务的机构，包括第三方支付机构、银行等。
